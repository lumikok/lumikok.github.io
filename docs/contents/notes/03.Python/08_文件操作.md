##  1_函数常用参数

[返回目录](#目录).

**注意**：主要针对**操作文本文件**。

**通用核心参数(前三必写)**：

- **file/文件路径**：所操作文件的相对路径 / 绝对路径；
- **mode**：文件操作模式（具体见 [2_文件操作模式](#2_文件操作模式)）；
- **encoding**：编码，文本文件一般统一用 `utf-8`，防止中文乱码
- **errors**：编码错误处理，`ignore` 忽略、`replace` 替换
- **newline**：控制换行符，跨平台统一格式

## 2_文件操作模式

[返回目录](#目录).

下表总结了部分常用 `mode`：

| 模式         | 功能    | 特点            |
| ---------- | ----- | ------------- |
| `r`        | 只读    | 文件不存在报错       |
| `w`        | 只写    | 覆盖写入，不存在则新建   |
| `a`        | 追加    | 末尾追加内容，不存在则新建 |
| `r+`       | 读写    | 从文件开头读写，不存在报错 |
| `w+`       | 读写    | 清空后读写，不存在则新建  |
| `a+`       | 读写    | 末尾追加读写，不存在则新建 |
| `rb/wb/ab` | 二进制读写 | 图片、视频、音频、压缩包  |

**补充**：由表可知，**有读则文件必存在（除了r+ 读写）**。

## 3_文件基础操作

### 3_1_Open_and_Close

[返回目录](#目录).

python 打开与关闭文件的具体语法如下：

```python
# 打开文件
f = open("text.txt","r",encoding = "utf-8")

...
...
# 手动关闭，释放资源（必须）
f.close()
```

**注意**：当操作完目标文件后，**必须关闭文件**，防止当前程序占用资源导致其他程序无法调用。由此可见，有关资源释放的操作比较繁琐，因此下面介绍一种**实际开发常用的释放资源的操作方式**。

具体代码如下：

```python
with open("text.txt","r",encoding = "utf-8") as f:
	# 写具体的读写逻辑
	pass # 先占位，后续写具体逻辑
```

**说明**： with 是 python 中的一个上下文管理器，常用来标识作用域，用在此处可实现自动关闭文件、释放资源（**无需 close 语句**）。

### 3_2_Read

[返回目录](#目录).

python 读取文件的具体语法如下：

```python
with open("text.txt","r",encoding = "utf-8") as f:
	content = f.read() # 读取所有内容
	f.readline() # 每次调用读取一行
	content_list = f.readlines() # 读取所有行，返回列表，循环输出即可
	for line in content_list:
		print(line.strip()) # strip()去掉字符串两端多余的内容
```

**注意**：在不重置文件指针的情况下，读取仅可读一次（读完后，指针已经停留在了文件末尾）。

### 3_3_Write

[返回目录](#目录).

python 写文件的具体语法如下：

```python
# 1. 覆盖写入
with open("test.txt", "w", encoding="utf-8") as f:
	f.write("第一行\n") # 传入字符串，写单行
	f.writelines(["第二行\n","第三行"]) # 传入列表，写多行
	
# 2. 追加写入
with open("test.txt", "a", encoding="utf-8") as f:
	f.write("\n追加新内容") # 仅调整操作模式即可
```

## 4_文件指针操作

[返回目录](#目录).

python 常用文件指针操作具体语法如下：

```python
with open("test.txt", "r+", encoding="utf-8") as f:
	print(f.tell()) # 获取当前指针位置
	f.seek(0) # 移动指针到文件开头
```

## 5_常用文件工具

[返回目录](#目录).

python 中可以引入 `os` 模块来便携操作文件本身，具体语法如下：

```python
import os
os.path.exists("test.txt") # 判断文件是否存在
os.rename("旧名.txt","新名.txt") # 重命名
os.remove("test.txt") # 删除文件
os.mkdir("folder") # 创建文件夹(folder)
```

## 6_JSON文件操作

为存储结构化数据和跨语言跨平台通用，python 中一般操作 JSON 文件，其需要引入 json 模块，语法为：`import json`。

### 6_1_序列化-write

[返回目录](#目录).

python 中写入 JSON 文件的具体语法如下：

```python
import json

data = {"name":"Python","age":"18"}
with open("data.json","w",encoding="utf-8") as f:
	json.dump(data,f,ensure_ascii=False,indent=4) # 序列化操作
```

**说明**：**`json.dump(...)`**，序列化操作，参数中 **`data`和`f`填**，如数据含有**中文**，需取消 **`ensure_ascii`(默认为True)**，**`indent`** 为缩进参数(用于格式化内容，更直观)。

### 6_2_反序列化-read

[返回目录](#目录).

python 中读取 JSON 文件的具体语法如下：

```python
import json

with open("data.json","r",encoding="utf-8") as f:
	data = json.load(f)
	print(data) # 字典类型
```
