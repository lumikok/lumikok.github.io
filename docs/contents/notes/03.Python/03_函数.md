### 函数定义与调用

- **介绍**：函数是组织好的、可重复使用的、用来实现特定功能的代码片段
- 定义与调用：
```python
# 定义函数
def 函数名(参数列表):
	函数体
	······
	return 返回值
	
# 调用函数
函数名(参数)
```

**注意**： 函数中通过缩进来描述归属关系。

### 参数与返回值

**注意**：
- 与 `C++` 大致相同。
- 返回值可以使用**组包和解包**，即**可返回多个值**。

### 函数说明文档

- **介绍**：本质**注释**，用三个引号包裹，用于解释函数的功能、参数、返回值等信息，便于理解、使用和维护。
- **案例**：
```python
def cal_area(l,r):  
    """  
  
    :param l: 长  
    :param r: 宽  
    :return: 面积  
    """    return l*r  
  
help(cal_area)  
m = cal_area(5,3)  
print(m)
```

- **查看方式**：
	- 使用 `help` 函数
	- 鼠标悬浮在函数上，自动展示

### 函数嵌套调用

- **介绍**： 嵌套调用指的是在一个函数中，又调用了另外一个函数
- **特征**：遵循栈结构，最后被调用的函数最先返回（**后进先出**）
- **案例**：

```python
def print_a():  
    print("a------")  
    print_b()  
    print("a------")  
def print_b():  
    print("b------")  
    print_c()  
    print("b------")  
def print_c():  
    print("c------")  
  
print_a()
```

def sanjiao_area(d,h):

    """

    :return: 要求的内容

### 变量作用域

- **全局变量**：在函数外定义的变量，在整个文件中都可以使用
- **局部变量**：在函数内定义的变量，只能在函数内使用，外部无法访问（函数执行完毕后，会自动销毁其内部局部变量）
- **`global` 关键字**：明确告诉 `Python` 解释器，在函数中要使用全局变量，从而可以在函数内部修改全局变量的值
- **注意**：
	- 尽量避免在函数中使用全局变量
	- 考虑使用函数参数和返回值来传递数据，而不是依赖全局变量
	- `global` 主要用在程序的状态、配置和计数器等场景中

### 传参方式

- **位置参数**：根据函数定义时的位置传递参数（**注意**：顺序需要完全一致）
- **关键字参数**：以函数定义时形参名称作为关键字，以 “键=值” 的形式来传递参数（无顺序要求）
- **注意**：二者混用时 -> 关键字参数必须在位置参数之后

### 默认参数

- **介绍**：默认参数也称为缺省参数，用于在定义函数时，为参数提供**默认值**，调用函数时，可以不传递有默认值的参数
- **注意**：
	- 默认参数必须放在其他参数之后，默认参数可以存在多个
	- 函数调用时，如果为默认参数传递了值，则以传递值为准

### 不定长参数

- **介绍**：不定长参数也叫可变参数，用于函数定义及调用时参数个数不确定的场景
- **类型**：
	- **位置传递**：
```python
def calc_data(*args):
	
	
data = calc_data(10,20,30)
print(data)

data = calc_data(1)
print(data)
```

- **注意**：
	- 传递的所有匹配的位置参数都会被 `args` 变量收集，这些参数会合并封装成一个元组（**不会封装关键字参数**）
	- `args` 非固定，可为任何合法的变量名

- **关键字传递**：
```python
def calc_data(*args,**kwargs):
	min_data = min(args)
	max_data = max(args)
	
	if kwargs.get('round'):
		min_data = round(min_data,kwargs.get('round'))
	
	return min_data,max_data
	
data = calc_data(100,200,300,round=2,count=0)
print(data)

data = calc_data(100,200,300)
print(data)
```

**注意**：这些 “键=值” 参数都会被 `kwargs` 接受，并合并封装成一个**字典类型**（注意字典的解包方式 -> `**`）

- 位置传递 -> 数量不确定
- 关键字传递 -> 选项不确定（**定制函数的行为**）

### 函数作为参数

**例子**：
```python
def add(x,y):
	return x+y

def subtract(x,y):
	return x-y

def calc(x,y,oper):
	return oper(x,y)

result = calc(10,20,add)
print(result)
```

**补充**：这里的 `add` 传递的是函数中封装的逻辑

### 匿名函数

- **介绍**：匿名函数指的是没有名称的函数，需要通过 `lambda` 表达式来声明函数，简化为单行表达式
- **案例**：
```python
lambda 参数列表 : 函数体

out_line = lambda : print("    ")
add = lambda x,y : x + y

out_line()
print(add(100,200))
```
**注意**：函数逻辑比较简单（一步操作）且只在一个地方使用时，可以考虑使用匿名函数，简化书写（**通常作为高阶函数的参数使用**）

### 案例

```python
def cal_order_cost(*args,cupon=0,score=0,express=0):  
    """  
    根据传入的一批商品信息、优惠、运费信息计算订单的总金额  
    :param args: 商品信息  
    :param cupon:优惠券  
    :param score:积分抵扣  
    :param express:运费信息  
    :return:订单总金额  
    """    total_price = [goods[1] * goods[2] for goods in args]  
    total_cost = sum(total_price)  
  
    if total_cost >= 5000 and cupon <= total_cost:  
        total_cost -= cupon  
  
    if total_cost >= 5000 and score // 100 <= total_cost:  
        total_cost -= score // 100  
    total_cost += express  
    return total_cost  
  
# 测试  
  
cost = cal_order_cost(("鼠标",100,3),("键盘",200,3),("手机",3999,1),cupon=10,score=4000,express=10)  
print(cost)
```
