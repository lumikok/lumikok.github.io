## 基本操作

- **定义**：字符串是字符的容器，类似于 `C++` 中的 `string` 类，可存储任意数量的字符。
- **特性**：不可变性（**无法修改**），有序性，可迭代性。
- **访问方式**：与 `list` 一样，下标访问
- **字符串的切片**：与 [[01_list#^b2816|`list`]] 一样

## 常用方法

| 方法                        | 作用                   | 样例                        |
| ------------------------- | -------------------- | ------------------------- |
| find()                    | 查找子串，返回第一次出现的索引位置    | s.find('python')          |
| count()                   | 统计子串在字符串中出现的次数       | s.count('h')              |
| upper()                   | 所有字母转换成大写            | s.upper()                 |
| lower()                   | 所有字母转换成小写            | s.lower()                 |
| split()                   | 将字符串按指定分隔符分割成列表      | s.split(' ')              |
| strip()                   | 去除字符串两端的空白字符或指定字符    | s.strip() or s.strip('s') |
| replace()                 | 把指定子串替换为新的子串         | s.replace('c','a')        |
| startswith()              | 检查字符串是否以指定子串开头，返回布尔值 | s.startswith('c')         |

补：`endswith()` 与 `startswith()` 对应
