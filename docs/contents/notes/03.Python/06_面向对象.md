## 概述

- **面向过程编程**：
	- **核心思想**：把一个需求分解成一系列要执行的步骤，然后按照步骤依次执行这些任务（**关注的是流程、步骤**）
	- **适用场景**：面向过程编程非常直接，适合简单、线性的任务
- **面向对象编程**：
	- 对象可以理解为现实中具体的物体（**万物皆对象**）
	- 它把一个人/物的特征和功能打包到一起，是面向对象编程的基本单元（**关注的是谁来帮我做这件事**）
	- **类与对象**：
		- **类**：描述的是一组具有相同属性（特征）和方法（功能/行为）的模版
		- **对象**：对象是类的实例，是基于类创建出来的（实例对象）
**注意**：对象是由类创建出来的，创建对象的过程，也称为对象的实例化，一个类可以创建无数个对象

## 类与对象

### 基础结构

**定义类的语法如下**：
```python
# 定义类
class 类名:
	pass

# 创建对象
对象名 = 类名()
对象名.属性名1 = 属性值1
对象名.属性名2 = 属性值2

class Car:
	pass
c1 = Car()
c1.brand = "BMW"
c1.name = "X5"
c1.price = 500000
print(c1.__dict__) # 会将对象中的所有属性以字典的形式输出出来
```

**注意**：
- **类名的命名规范**：每个单词的首字母都是大写的，单词之间无间隔
- **`__dict__`**：`Python` 自定义类实例的一个特殊属性，用于以字典形式存储对象的属性

### 类中方法的定义

**语法如下**：

```python
# 定义类
class 类名:
	def __init__(self,参数列表)
		self.属性名 = 参数值
		self.属性名 = 参数值

对象名 = 类名(参数列表)

# 定义类
class Car:
	def __init__(self,c_brand,c_name,c_price):
		self.brand = c_brand
		self.name = c_name
		self.price = c_price
		
# 创建对象
c1 = Car("BMW","X5",5000000)
print(c1.__dict__)
```

**注意**：
- 定义在类的外面的称之为函数，定义在类中的函数称之为**方法**
- **`self`**：方法的第一个参数，表示当前创建的实例对象
- **`__init__`**：初始化方法，**对象创建后自动调用**，主要用于设置对象的初始状态

## 实例方法

实例方法，本质函数，与函数定义的方式一致
**不同之处**：都多一个 **`self`** 表示当前创建的实例对象

**示例代码如下**：

```python
class Car:  
    def __init__(self,brand,name,price):  
        self.brand = brand  
        self.name = name  
        self.price = price  
  
    def running(self):  
        print(f"{self.brand} {self.name} 正在高速行驶...")  
    def total_cost(self,discount,rate):  
        return self.price * discount + self.price * rate  
  
c1 = Car("BMW","X5",500000)  
total_cost = c1.total_cost(0.9,0.1)  
print(f"提车总价为：{total_cost:.0f}")  
  
c1.running()
```

## 魔法方法

- **介绍**：魔法方法是指 `Python` 中提供的以**双下划线**开头和结尾的特殊方法，用于定义类的特殊行为，比如：`__init__`
- **注意**：魔法方法不需要手动调用，`Python` 会在合适的时机**自动调用**

**常见魔法方法如下**：

| **魔法方法**                            | **描述**                     |
| ----------------------------------- | -------------------------- |
| `__init__`                          | 初始化方法                      |
| `__str__`                           | 字符串表示的方法                   |
| `__eq__`                            | 比较两个对象是否相等                 |
| `__lt__`,`__le__`,`__gt__`,`__ge__` | 比较两个对象的大小（小于，小于等于，大于，大于等于） |

**代码示例如下**：

```python
class Car:  
    def __init__(self,brand,name,price):  
        self.brand = brand  
        self.name = name  
        self.price = price  
  
    def running(self):  
        print(f"{self.brand} {self.name} 正在高速行驶...")  
    def __str__(self):  
        return f"{self.brand} {self.name} {self.price}"  
    def __eq__(self,other):  
        return self.brand == other.brand and self.name == other.name and self.price == other.price  
    def __lt__(self,other):  
        return self.price < other.price  
  
c1 = Car("BMW","X5",500000)  
print(c1)  
c2 = Car("BMW","X5",500000)  
  
print(c1 == c2)  
print(c1 < c2)  
  
c1.running()
```

**总结**：
- **`print(c1)`**：默认输出对象的内存地址（ 16 进制）
- **``print(c1==c2)``**：默认是基于对象的内存地址进行比较
- **`print(c1 < c2)`**：默认自定义的对象之间不可以进行大小比较
- **魔法方法的使用**：上述代码的运行结果将与**魔法方法的返回值**挂钩
**补充**：类似于 `C++` 中的**运算符重载**

## 实例属性与类属性

- **属性**：
	- **实例属性**：实例属性属于每个具体对象的属性，每个对象都是独立的（各个对象特有的数据）
	- **类属性**：类属性是属于类本身的属性，所有实例共享的（所有对象共享的数据或配置）

**示例代码如下**：

```python
class Car:

	# 类属性
	wheel = 4
	tax_rate = 0.1
	
	# 实例属性
	def __init__(self,c_brand,c_name,c_price):
		self.brand = c_brand
		self.name = c_name
		self.price = c_price
		
```

**注意**：通过实例查找属性时，会先查找实例属性，实例属性不存在时，再查找类属性（故存在**覆盖**情况）
**操作方式**：
- **类属性**：通过 **`类名.属性`**
- **实例属性**：通过 **`实例对象.属性`**

## 案例

**具体要求如下**：

采用面向对象的编程思想，完成教务管理系统的开发。教务管理系统可以管理在校学生的成绩信息，通过控制台菜单与用户交互，具体的功能如下：

1. **添加学生成绩**：根据输入的学生姓名、语文成绩、数学成绩、英语成绩，记录在系统中
2. **修改学生成绩**：根据输入的学生姓名，修改对应的学生成绩
3. **删除学生成绩**：根据输入的学生姓名，删除对应的学生成绩
4. **查询指定学生成绩**：根据输入的学生姓名，查找对应的学生成绩，并输出
5. **展示全部学生成绩**：展示出系统中所有学生的成绩

**具体代码如下**：

```python
# 学生类，存储每个学生的具体信息  
class Students():  
    def __init__(self,name,cscore,mscore,escore):  
        self.name = name  
        self.cscore = cscore  
        self.mscore = mscore  
        self.escore = escore  
    def __str__(self):  
        m = self.mscore + self.escore + self.cscore  
        return f"姓名：{self.name}，语文成绩：{self.cscore}，数学成绩：{self.mscore}，英语成绩：{self.escore}，总分：{m}"  
    def get_name(self):  
        return self.name  
    def set_score(self,chinese,math,english):  
        self.cscore = chinese  
        self.mscore = math  
        self.escore = english  
  
# 学生成绩管理系统类  
class ScoreManager:  
    def __init__(self):  
        self.student = {} # 字典 存储 key = name value = students类  
  
    def add_student(self):  
        print("===== 添加学生成绩 =====")  
        name = input("请输入学生姓名：")  
        if name in self.student:  
            print("该学生已存在，添加失败！\n")  
            return  
        chinese = int(input("请输入语文成绩："))  
        math = int(input("请输入数学成绩："))  
        english = int(input("请输入英语成绩："))  
  
        student = Students(name,chinese,math,english)  
        self.student[name] = student  
        print("添加成功！\n")  
  
    def update_score(self):  
        print("===== 修改学生成绩 =====")  
        name = input("请输入学生姓名：")  
        if name not in self.student:  
            print("该学生不存在，修改失败！")  
            return  
        chinese = int(input("请输入新语文成绩："))  
        math = int(input("请输入新数学成绩："))  
        english = int(input("请输入新英语成绩："))  
        self.student[name].set_score(chinese,math,english)  
  
    def delete_student(self):  
        print("===== 删除学生成绩 =====")  
        name = input("请输入学生姓名：")  
        if name not in self.student:  
            print("该学生不存在，删除失败！")  
        else:  
            del self.student[name] # 将 键 和 值 一起删掉了  
            print("删除成功！")  
  
    def find_student(self):  
        print("===== 查询学生成绩 =====")  
        name = input("请输入学生姓名：")  
        student = self.student.get(name) # 获取 键 对应的 值（students类）  
        if student:  
            print(student)  
        else:  
            print("未查询到该学生！")  
  
    def show_all(self):  
        print("===== 全部学生成绩 =====")  
        if self.student:  
            for name,student in self.student.items(): #获取所有键值对  
                print(student)  
        else:  
            print("暂时没有学生信息！")  
  
    def show_menu(self):  
        while True:  
            print("========== 教务成绩管理系统 ==========")  
            print("1. 添加学生成绩")  
            print("2. 修改学生成绩")  
            print("3. 删除学生成绩")  
            print("4. 查询指定学生成绩")  
            print("5. 展示全部学生成绩")  
            print("0. 退出系统")  
  
            choice = int(input("请输入选择："))  
  
            match choice:  
                case 1:  
                    self.add_student()  
                case 2:  
                    self.update_score()  
                case 3:  
                    self.delete_student()  
                case 4:  
                    self.find_student()  
                case 5:  
                    self.show_all()  
                case 0:  
                    print("系统成功退出！")  
                    break  
                case _:  
                    print("错误！请重新选择 \n")    

# 主函数入口
if __name__ == "__main__":  
    score_manager = ScoreManager()  
    score_manager.show_menu()
```

**总结**：

1. **类与对象结构**
	- **`Students` 类**：**数据类**，负责封装单个学生的姓名、三科成绩，提供 `__str__`、`get_name`、`set_score` 方法。
	- **`ScoreManager` 类**：**管理类**，负责实现增删改查、菜单交互、数据存储，是系统核心逻辑类。
	- **面向对象原则**：**数据与业务分离**，学生类存数据，管理类做操作。
2. **字典存储核心**
	- **存储容器**：`self.student = {}`（字典）。
	- **键（key）**：学生姓名（字符串），用于**快速查找、去重、判断存在**。
	- **值（value）**：`Students` 实例对象，内部包含**姓名 + 语文 + 数学 + 英语**全部数据。
	- **关键特点**：键和对象内都存储姓名，**键用于检索，姓名用于展示**，不冲突。
