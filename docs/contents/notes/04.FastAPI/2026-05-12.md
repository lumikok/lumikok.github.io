# 第一步

## FastAPI 文件启动方式

FastAPI 文件的启动方式有：

1. `fastapi dev`：官方推荐，适合开发，自动热重载，自带文档 `.../docs`
2. `uvicorn main:app --reload`：标准启动，`main`-文件名，`app` 对应的 `FastAPI()` 实例，`--raload` 自动热重载

遇到的问题：
使用第一个的时候，ide 没有下载完整的 FastAPI，所以需要 `pip install "fastapi[standard]"`。

ctrl + c 终止运行

 `http://127.0.0.1:8000`：默认访问地址
 `http://127.0.0.1:8000/docs`：自动生成的交互式 API 文档

## 最简程序

```python
from fastapi import FastAPI

app = FastAPI()

@app.get("/")
async def root():
	return {"message": "Hello"}
```

1. FastAPI 是一个类，能给你的 API 提供所有功能
2. app 是 FastAPI 的一个实例
3. 定义一个路径操作装饰器，写 HTTP 方法，写路径“第一个/后的”。
4. 后面跟的就是这个装饰器的函数（具体实现的功能）

# 路径参数

## 路径操作

路径操作是按照顺序执行的，所以需要注重参数的顺序，比如先获取用户 id，然后把 id作为路径参数，来获取指定用户的数据。

同样，对同一路径参数的操作只能有一个（默认只执行第一个）。

## 预设值

路径参数可预设，用 `Enum` 类

```python
from fastapi import FastAPI  
from enum import Enum  
  
class basedata(str,Enum):  
    name = "name"  
    age = "age"  
    sex = "sex"  
  
app = FastAPI()  
  
@app.get("/root/{id}")  
def get_data(id: basedata):  
    if id.name == "name":   # 比较方式一
        return {"name":"jon"}  
    elif id.value == "age":  # 比较方式二
        return {"age":18}  
    elif id == basedata.sex:   # 比较方式三
        return {"sex":"male"}  
    return None
```

**注意**：

- 枚举成员**本身是枚举对象**，赋值的字符串只存在对象的 `.value` 属性里。
- 前端路径传参永远是**普通字符串**。
- 函数参数标注枚举类型时，FastAPI 会**自动把请求字符串转为枚举对象**传入函数。
- 原生 `Enum` 枚举对象不能直接和字符串 `==` 比较（方式一），只能用 `对象.value` 取值对比。
- 写成 `class 枚举类(str, Enum)` 后，枚举对象**兼容字符串特性**，可直接和字符串相等比较，适配 FastAPI 自动校验。
1. 必须先继承 `str`，再继承 `Enum`：FastAPI 需要把**路径参数自动转成字符串**再匹配枚举，传入函数的本质为枚举类，但是继承了 `str`，具有字符串的行为能力
2. 枚举类成员的本质是实例对象

### 返回枚举成员

可以从路径操作里返回枚举成员

返回给客户端时，会自动转换成对应的值

```python
from fastapi import FastAPI  
from enum import Enum  
  
class Basedata(str,Enum):  
    name = "name"  
    age = "age"  
    sex = "sex"  
  
app = FastAPI()  
  
@app.get("/data/{id}")  
def get_data(id: Basedata):  
    if id.name == "name":  
        return {"id_name": id,"message": "lov"}  
    elif id.value == "age":  
        return {"id_name": id,"message": 18}  
    elif id == Basedata.sex:  
        return {"id_name": id,"message": "man"}  
    return None
```

返回的 JSON 数据：

```json
{
  "id_name": "name",
  "message": "lov"
}
```

## 包含路径的路径参数

FastAPI 底层完全遵循 OpenAPI 规范，本身不支持

但是可以用 Starlette 内置工具（选项声明）来实现，并且不会影响文档

路径转换器，直接使用 Starlette 的选项声明包含路径的路径参数：

```python
/files/{file_path:path}
```

`:path` 说明该参数应匹配路径。

```python
from fastapi import FastAPI  
from enum import Enum  
  
class Basedata(str,Enum):  
    name = "name"  
    age = "age"  
    sex = "sex"  
  
app = FastAPI()  
  
@app.get("/files/{file_path:path}")  
async def read_file(file_path: str):  
    return {"file_path": file_path}
```

# 查询参数

声明的参数不是路径参数时，路径操作函数就默认它为查询参数。

```python
from fastapi import FastAPI

app = FastAPI()

fake_items_db = [{"item_name": "Foo"}, {"item_name": "Bar"}, {"item_name": "Baz"}]


@app.get("/items/") 
async def read_item(skip: int = 0, limit: int = 10):
    return fake_items_db[skip : skip + limit] # 切片
```

访问方式：
	查询参数是键值对集合，其都写在 URL？之后，多个键值对用 `&` 连接。

如：

```
http://127.0.0.1:8000/items/?skip=0&limit=10
```

## 默认值

默认值可修改，不传的参数都取默认值（可以一个都不写，写了必须写 `?`）。

## 可选参数

默认值设为 None，即为可选参数（可以不提供）

```python
from fastapi import FastAPI

app = FastAPI()


@app.get("/items/{item_id}")
async def read_item(item_id: str, q: str | None = None):
    if q:
        return {"item_id": item_id, "q": q}
    return {"item_id": item_id}
```

可知，路径参数可以和查询参数混用，可以自动识别

## 设置为 bool 类型

查询参数可以设置成 bool 类型，比如：

```python
from fastapi import FastAPI

app = FastAPI()


@app.get("/items/{item_id}")
async def read_item(item_id: str, q: str | None = None, short: bool = False):
    item = {"item_id": item_id}
    if q:
        item.update({"q": q})
    if not short:
        item.update(
            {"description": "This is an amazing item that has a long description"}
        )
    return item
```

访问可用 `...short=1` / `...short=true` / `...short=True` / `...short=on` / `...short=yes` 及其任意大小写形式。FastAPI 可以自动转换参数类型。

## 补充

可以设置多个路径参数和查询参数，声明顺序不重要，主要关注的是参数名。

必选查询参数就是不设默认值，设默认值是非必选，设为 `None` 就是可以没有这个参数。
