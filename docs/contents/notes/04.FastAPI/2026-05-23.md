# 声明请求示例数据

## 为什么要声明请求示例

主要是给 API 的使用者提供信息用的，方便对接。

比如：前后端对接 API。

## 实现方式

### Pydantic 模型

声明 `examples`：

```python
class Item(BaseModel):
	name: str
	... # 其它参数
	model_config = {
		"json_schema_extra": {
			"examples": [
				{
					"name": "lov"
					...
				}
			]
		}
	}
```

### Field

Pydantic 中使用 `Field() ` 时，同样可以声明额外的 `examples`：

```python
name: str = Field(examples=["lov"])
```

### 其他情况

`examples` 可以用于：

- `Path()`
- `Query()`
- `Header()`
- `Cookie()` 
- `Body()`
- `Field()`
-   ......

一个请求体可以有多个 `examples`：

```python
examples=[
	{
		...,
	},
	{
		...,
	},
	...
],
```

# 额外数据类型

此部分，用到了，再去查文档就行了

- `UUID`：
	- 用于标识唯一的资源，在数据库和系统中用作 ID
	- 以 `str` 表示
- `datetime.datetime`：
	- 处理时间（带时区信息），python 类
	- ISO 8601 格式的 `str`，比如：`2026-05-23T14:30:00+08:00`
- `datetime.date`：
	- 同上，`2026-05-23`
- `datetime.time`：
	- 同上，`14:30:00.003`（时 : 分 : 秒 : 微秒）
- `datetime.timedelta`：
	- 同上，表示为 `float`，代表持续总秒数
- `frozenset`：
	- 在请求和响应中，被当做 `set` 处理：
		- 请求：列表被读取，消掉重复元素，组成一个 `set`
		- 响应：`set` 被转换成 `list`
		- 生成的 JSON Schema 包含 `uniqueItems=true`，表示值唯一
- `bytes`:
    - 标准的 Python `bytes`。
    - 在请求和响应中被当作 `str` 处理。
    - 生成的 JSON Schema 将指定这个 `str` 是二进制格式。
- `Decimal`:
    - 标准的 Python `Decimal`。
    - 在请求和响应中被当做 `float` 一样处理。

# Cookies

**[计算机网络]**    什么是 Cookie？

Cookie 是服务器发送到用户浏览器并保存在本地的一小块数据，用于下一次向同一服务器发送请求时，不用再重复获取数据。是**HTTP 协议**的一部分。

**核心**：记住信息，比如登录状态、用户行为。

## 导入

```python
from fastapi import Cookie
```

## 声明

```python
ads_id: str | None = Cookie(default=None)
```

推荐使用：

```python
ads_id: Annotated[str | None, Cookie()] = None
```

使用 `Cookie` 声明 cookie 参数的方式与 `Query` 和 `Path` 相同。

# Header 参数

> 补网络知识：请求头

Header声明方式和与 Cookie 基本一样。

## 自动替换

但需**注意**的是：

大部分标准请求头用**连字符**分隔，比如：`user-name`，但是 Python 中不允许这样的变量，而 FastAPI 解决了这个问题，`Header` 能够自动替换 → “把 `user_name` 自动转换为 `user-name`”。

并且 HTTP 请求头不区分大小写

如果不想要自动转换，可以把 `Header` 的 `convert_underscores` 参数设置为 `False`。

**但是，不建议禁用**。

同样，类型可以声明为 `list[str]` 这样的容器，以便于接受重复的请求体，比如：

```python
x_name: Annotated[list[str] | None, Header()] = None
```

与路径操作通信时，可以发送多个 HTTP 请求头：

```http
X-Name: lov
X-Name: nov 
```
