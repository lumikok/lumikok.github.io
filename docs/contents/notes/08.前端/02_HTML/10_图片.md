# 图片



## 添加图片



在 HTML 中，可以使用 `<img>` 标签来添加图片。`<img>` 标签是一个自闭合标签，通常需要指定 `src` 属性来定义图片的路径，以及 `alt` 属性来提供图片的替代文本：

```html
<img src="图片路径" alt="替代文本" />
```

- `src` 属性：指定图片的路径，可以是相对路径或绝对路径。
- `alt` 属性：提供图片无法显示时的替代文本，有助于提高网页的可访问性。

### 设置图片大小


可以使用 `width` 和 `height` 属性来设置图片的宽度和高度：

```html
<img src="图片路径" alt="替代文本" width="300" height="200" />
```

### 图像标题



可以使用 `title` 属性为图片添加标题，当用户将鼠标悬停在图片上时，会显示该标题：

```html
<img src="图片路径" alt="替代文本" title="图片标题" />
```

## 版权许可



在使用图片时，请确保您有权使用该图片。许多图片受版权保护，未经许可使用可能会导致法律问题。建议使用以下几种方式获取合法的图片：

- 使用公共领域或 Creative Commons 许可的图片。
- 使用您自己拍摄或创建的图片。
- 从合法的图片库购买或下载图片。
- 遵守图片的使用条款和条件。
- 为图片添加适当的版权信息和归属声明。

## 说明文字



可以使用 `<figure>` 和 `<figcaption>` 标签来为图片添加说明文字：

```html
<figure>
  <img src="图片路径" alt="替代文本" />
  <figcaption>这是图片的说明文字。</figcaption>
</figure>
```

## CSS 背景图片



除了使用 `<img>` 标签外，还可以通过 CSS 来设置背景图片：

```css
body {
  background-image: url("图片路径");
  background-size: cover; /* 使背景图片覆盖整个容器 */
  background-repeat: no-repeat; /* 防止背景图片重复 */
}
```

CSS 背景图片适用于网页的整体背景或特定元素的背景，主要用于装饰性目的。
