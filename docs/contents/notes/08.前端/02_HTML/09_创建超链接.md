# 超链接

## 什么是超链接



超链接（Hyperlink）是网页中的一个元素，允许用户通过点击链接跳转到另一个网页、文件或资源。超链接通常以文本或图像的形式出现，用户点击后会被引导到指定的目标位置。

## 创建超链接



通过将文本或其他内容包裹在 `<a>`（anchor）标签中，并给它添加 `href` 属性，可以创建一个超链接。`href` 属性指定了链接的目标地址：

```html
<a href="https://www.example.com">访问示例网站</a>
```

### 块级链接



任何内容，甚至块级元素，都可以作为链接出现，比如标题元素：

```html
<a href="https://www.example.com">
  <h2>访问示例网站</h2>
</a>
```

### 图片链接



图片也可以作为超链接的内容：

```html
<a href="https://www.example.com">
  <img src="image.jpg" alt="示例图片" />
</a>
```

### 添加补充信息



可以使用 `title` 属性为超链接添加补充信息，当用户将鼠标悬停在链接上时，会显示该信息：

```html
<a href="https://www.example.com" title="点击访问示例网站">访问示例网站</a>
```

### 打开新窗口



默认情况下，点击超链接会在当前窗口打开目标页面。如果希望在新窗口或新标签页中打开链接，可以使用 `target` 属性：

```html
<a href="https://www.example.com" target="_blank">在新窗口打开示例网站</a>
```

## 统一资源定位符（URL）与路径（Path）



统一资源定位符（URL）用于指定网页或资源的地址。URL 可以是绝对路径或相对路径：

- 绝对路径：包含完整的地址信息，例如 `https://www.example.com/page.html`。
- 相对路径：相对于当前页面的位置，例如 `page.html` 或 `../images/pic.jpg`。

### 示例


```html
<!-- 绝对路径 -->
<a href="https://www.example.com/page.html">访问示例页面</a>
<!-- 相对路径 -->
<a href="page.html">访问当前目录下的页面</a>
<a href="subdir/page.html">访问子目录中的页面</a>
<a href="../images/pic.jpg">访问上级目录中的图片</a>
---
```

### 文档片段



超链接除了可以链接到文档外，也可以链接到文档中的特定位置。要实现这一点，必须首先给要链接到的位置添加一个 `id` 属性：

```html
<h2 id="section1">章节 1</h2>
<p>这是章节 1 的内容。</p>
<a href="index.html#section1">跳转到章节 1</a>
```

在上面的例子中，点击链接会跳转到 `index.html` 页面中的 `section1` 位置。

## 链接非 HTML 文件



超链接不仅可以链接到 HTML 页面，还可以链接到其他类型的文件，如 PDF、Word 文档、图片等：

```html
<a href="document.pdf">下载 PDF 文档</a>
<a href="image.jpg">查看图片</a>
<a href="file.docx">下载 Word 文档</a>
```

### 下载链接



如果希望用户点击链接时直接下载文件而不是在浏览器中打开，可以使用 `download` 属性：

```html
<a href="document.pdf" download="document.pdf">下载 PDF 文档</a>
```

## 电子邮件链接



当点击链接时，可以打开用户的默认电子邮件客户端，并自动填写收件人地址。使用 `mailto:` 协议来创建电子邮件链接：

```html
<a href="mailto:someone@example.com">发送电子邮件</a>
```

## 电话链接



类似地，可以创建电话链接，点击后会启动拨号应用程序。使用 `tel:` 协议来创建电话链接：

```html
<a href="tel:+1234567890">拨打电话</a>
```
