# Head

头部元素里所包含的信息不会直接显示在网页内容中。它包含了网页的元数据，比如标题，指向 CSS 的链接，指向自定义网页图标的链接等其他对网页有用的信息。

## 添加标题

首先，要区分标题标签 `<title>` 和标题元素 `<h1>`。标题标签定义了网页的标题，显示在浏览器的标签页上，而标题元素定义了网页内容中的标题。

**具体区别：**

1. `<title>` 里的内容显示在标签页上，也就是那个浏览器顶部的标题栏里。（当然，添加书签时也会用到这个标题）
2. `<h1>` 里的内容显示在网页的正文部分，作为网页的主标题。

## 元数据

元数据就是描述数据的数据，它提供了关于网页的信息，但不会直接显示在网页内容中。元数据通常包含在 `<head>` 标签内的 `<meta>` 标签中。

### 常见的元数据类型

下面是一些常见的元数据类型

#### 指定字符编码

```html
<meta charset="UTF-8" />
```

这个标签指定了网页使用的字符编码为 UTF-8，这是一种支持多种语言字符的编码方式。（前面反复提到过的）

#### 添加描述

`<meta>`元素包含了`name`和`content`属性，用于提供网页的描述信息：

```html
<meta name="description" content="这是一个关于HTML头部元素的网页。" />
<meta name="author" content="你的名字" />
```

很明显，`name`属性指定了`meta`元素的类型，用于说明该元素包含了什么类型的信息。`content`属性指定了实际的元数据内容。

##### 此类描述信息的作用

1. **搜索引擎优化（SEO）**：搜索引擎会使用这些描述信息来理解网页的内容，从而在搜索结果中更好地展示网页。
2. **社交媒体分享**：当网页被分享到社交媒体平台时，这些描述信息通常会被用作预览文本，帮助用户了解网页内容。
3. **浏览器使用**：某些浏览器可能会使用这些描述信息来显示在标签页或书签中，帮助用户识别网页。
4. **辅助技术**：屏幕阅读器等辅助技术可能会使用这些描述信息来帮助视障用户理解网页内容。
5. **网页管理**：网站管理员可以通过这些描述信息更好地组织和管理网页内容。
6. **用户体验**：提供清晰的描述信息可以帮助用户更好地理解网页内容，提高用户体验。
7. **内容分类**：描述信息有助于对网页内容进行分类和索引，方便用户查找相关信息。
8. **广告投放**：广告平台可能会使用这些描述信息来确定适合投放的广告内容，提高广告的相关性和效果。
9. **内容推荐**：某些内容推荐系统可能会利用这些描述信息来推荐相关的网页内容给用户。

**以`description`为例**，它被使用在搜索引擎显示的结果页中，帮助用户了解网页的主要内容，从而决定是否点击进入该网页。

### 其他类型的元数据

当然，元数据的类型远不止这些，当查看网站源码时，你会发现更多不同类型的元数据标签，它们各自承担着不同的功能，帮助网页更好地与浏览器和搜索引擎进行交互。

### 自定义网页图标

网页图标（favicon）是显示在浏览器标签页上的小图标，可丰富网站设计，帮助用户快速识别网站。我们可以在元数据中添加对自定义图标的引用，以下是具体步骤：

1. **准备图标文件**：首先，准备一个图标文件，通常使用 `.ico`、`.png` 或 `.gif` 格式。图标的推荐尺寸为 16x16 像素或 32x32 像素,然后将其保存在与网站 HTML 文件相同的目录下，或者一个专门的 `images` 目录中。
2. **添加链接标签**：在 HTML 文件的 `<head>` 部分，使用 `<link>` 标签来引用图标文件。示例如下：

```html
<link rel="icon" href="favicon.ico" type="image/x-icon" />
```

这里，`rel="icon"` 指定了这是一个图标链接，`href` 属性指定了图标文件的路径，`type` 属性指定了图标的 MIME 类型。

**补充**：有时需要为不同设备和浏览器提供多种尺寸的图标，可以使用多个 `<link>` 标签。例如：

```html
<link rel="icon" href="favicon-16x16.png" sizes="16x16" type="image/png" />
<link rel="apple-touch-icon" href="apple-touch-icon.png" sizes="180x180" />
<link rel="icon" href="favicon-32x32.png" sizes="32x32" type="image/png" />
```

通过以上步骤，你就可以为你的网站添加一个自定义的网页图标，提升用户体验和网站的专业形象。

## 在 head 中添加 CSS 和 JavaScript

对于 CSS，通常使用 `<link>` 标签将外部样式表链接到 HTML 文件中，它有两个属性，`rel`表明这是一个样式表链接，`href`指定了样式表文件的路径。例如：

```html
<link rel="stylesheet" href="styles.css" />
```

对于 JavaScript，通常使用 `<script>` 标签将外部脚本文件链接到 HTML 文件中，它有一个属性，`src`指定了脚本文件的路径，但最好加一个`defer`属性，以确保脚本在 HTML 解析完成后再执行。例如：

```html
<script src="script.js" defer></script>
```

通过在 `<head>` 部分添加这些标签，可以确保网页在加载时正确应用样式和功能脚本。

## 设定语言

其实，很有必要为网页设定语言，这样浏览器和搜索引擎就能更好地理解网页内容。可以通过在 `<html>` 标签中添加 `lang` 属性来实现。例如：

```html
<html lang="zh-CN"></html>
```

这里，`lang="zh-CN"` 指定了网页的语言为简体中文（中国大陆）。根据需要，你可以将其更改为其他语言代码，如 `en`（英语）、`fr`（法语）等，同时也可以分段为不同语言内容设定不同的 `lang` 属性，以提高网页的可访问性和国际化支持。
