# 标题与段落

## 简述

在前文中已经介绍过了这部分内容，这里做一个简单的回顾：

- **段落**：使用 `<p>` 标签来定义，每段自动缩进
- **标题**：使用 `<h1>` 到 `<h6>` 标签来定义，`<h1>` 最大，`<h6>` 最小

## 层次结构示例

```html
<h1>红楼梦</h1>
<p>作者：曹雪芹</p>
<h2>第一回 甄士隐梦幻识通灵 贾雨村风尘怀闺秀</h2>
<p>满纸荒唐言，一把辛酸泪！都云作者痴，谁解其中味？</p>
<h3>第一节 大观园初成</h3>
<p>话说荣国府内，贾母正与众人议论大观园的事。</p>
<h4>第二节 宝玉初试云雨情</h4>
<p>宝玉与黛玉初次相见，情愫暗生。</p>
```

以此为例，**值得注意的是**：

- `<h1>`最好对每个页面只使用一次，作为该页面的主标题
- 确保层次结构是顺序正确的，不要跳跃使用标题标签，例如从 `<h2>` 直接跳到 `<h4>`
- 六个标题层次中，除非有必要，一般每页面不超过三级标题，如果可以的话建议将内容分散到多个页面中

## 语义

使用正确的标题和段落标签不仅有助于内容的组织和可读性，还能提升网页的语义结构，有利于搜索引擎优化（SEO）和辅助技术（如屏幕阅读器）的使用体验。

因此，我们虽然可以使用 CSS 来改变文本的外观，但应始终使用适当的 HTML 标签来表示内容的结构和意义。
