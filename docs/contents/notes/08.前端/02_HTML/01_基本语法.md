# 基本语法

## 简述 HTML 的基本规则

HTML 由一系列的**元素**组成，可以用来包装或标记网页内容的不同部分，使其具有特定的意义和功能。

扩展名为 `.html` 的文件被称为**HTML 文档**，其中最常见的是`index.html`，通常作为网站的首页。

## 元素

元素组成：

- **开始标签**（Opening Tag）：标记元素的开始，通常由尖括号包围的元素名称组成，例如 `<p>` 表示段落元素的开始。
- **内容**（Content）：位于开始标签和结束标签之间的实际内容，可以是文本、其他元素或两者的组合。
- **结束标签**（Closing Tag）：标记元素的结束，通常与开始标签类似，但在元素名称前加上斜杠，例如 `</p>` 表示段落元素的结束。

例如，下面是一个简单的段落元素：

```html
<p>这是一个段落。</p>
```

### 嵌套元素

元素可以嵌套在其他元素内部，以创建更复杂的结构。例如：

```html
<p>这是一个包含 <strong>加粗文本</strong> 的段落。</p>
```

在这个例子中，`<strong>` 元素嵌套在 `<p>` 元素内部，用于加粗文本。**但是注意**：元素关闭的顺序必须与打开的顺序相反。

### 空元素

并非所有元素都遵循元素的基本结构，有些元素是**空元素**，它们没有内容和结束标签，通常用于插入或者分隔内容。例如：

```html
<img src="image.jpg" alt="描述文本" />
<br />
<hr />
```

上面的 `<img>`、`<br>` 和 `<hr>` 都是空元素，用于插入图像、换行和制水平线。

## 属性

元素可以包含**属性**，用于提供额外的信息或配置选项。属性位于开始标签内，通常以键值对的形式表示。例如：

```html
<p class="intro">这是一个带有类属性的段落。</p>
```

在这个例子中，`class` 是属性名称，`intro` 是属性值。属性名称和值之间用等号连接，属性值通常用引号括起来。据此，我们可以知道属性应该包含：

- 一个在它和元素名之间的空格（多个属性之间也需要空格分隔）
- 属性名，后跟等号
- 属性值，通常用引号括起来（单引号或双引号均可，但不要混用）

### 添加属性

下面可以为`<img>`元素添加部分属性：

- `src`：指定图像的路径或 URL。
- `alt`：提供图像的替代文本，当图像无法显示时会显示该文本。
- `width` 和 `height`：指定图像的宽度和高度。
- `title`：为图像提供额外的信息，当用户将鼠标悬停在图像上时会显示该信息。

### 布尔属性

有些属性是**布尔属性**，它们不需要值。对于布尔属性，如果存在，则表示为真；如果不存在，则表示为假。例如：

```html
<input type="text" disabled />
```

在这个例子中，`disabled` 是一个布尔属性，表示该输入字段被禁用。

## HTML 文档

既然已经了解了 HTML 元素的基本结构，接下来我们来看元素如何组合成一个完整的 HTML 文档：

```html
<!DOCTYPE html>
<html>
  <head>
    <meta charset="UTF-8" />
    <title>示例 HTML 文档</title>
  </head>
  <body>
    <h1>欢迎来到我的网站</h1>
    <p>这是一个简单的 HTML 文档示例。</p>
  </body>
</html>
```

**剖析：**

1. `<!DOCTYPE html>`：声明文档类型，告诉浏览器这是一个 HTML5 文档。
2. `<html>`：根元素，包含整个 HTML 文档的内容。
3. `<head>`：包含文档的元数据，如字符编码、标题等。
4. `<meta charset="UTF-8" />`：指定文档的字符编码为 UTF-8。
5. `<title>`：定义文档的标题，显示在浏览器标签上。
6. `<body>`：包含网页的主要内容，如文本、图像等。
7. `<h1>`：表示一级标题。
8. `<p>`：表示段落。

当然，你现在也可以丰富这个文档，比如添加图片（`<img>`）或者链接（`<a>`）。

## 特殊字符

在 HTML 中，有些字符具有特殊的意义，例如 `<` 和 `>` 用于标记元素。如果你想在网页中显示这些字符，而不是让浏览器将它们解释为 HTML 代码，你需要使用**字符引用**。

下面是一些常用的字符引用：

| 字符 | 字符引用 |
| ---- | -------- |
| `<`  | `&lt;`   |
| `>`  | `&gt;`   |
| `&`  | `&amp;`  |
| `"`  | `&quot;` |
| `'`  | `&apos;` |

例如，如果你想在网页中显示 `<h1>`，你可以这样写：

```html
<p>要显示标题标签，请使用 &lt;h1&gt;。</p>
```

## 注释

注释用于在代码中添加说明或备注，浏览器不会显示注释内容。HTML 注释的语法如下：

```html
<!-- 这是一个注释 -->
```
