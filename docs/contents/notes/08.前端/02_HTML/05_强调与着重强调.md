# 强调与着重强调

在 HTML 中，强调和着重强调用于表示文本的重要性或语气的变化。HTML 提供了两个标签来实现这一点：`<em>` 和 `<strong>`。

## 强调

在口语表达中，当我们想要传达某个特殊的意义时，通常会通过语调的变化来实现。在书面表达中，HTML 使用 `<em>` 标签来表示这种强调。被 `<em>` 标签包裹的文本通常会以**斜体**显示，以示区别。

当然，这个斜体显示是带有语义的（屏幕阅读器会识别出并以不同的语调朗读），而不仅仅是视觉上的变化，因此不可随意使用。但当我们仅需要斜体效果时，可以用 `<i>` 标签，或者通过 CSS 来实现。

```html
<p>这是一个 <em>强调</em> 的例子。</p>
```

## 着重强调

在 HTML 中，`<strong>` 标签用于表示文本的**重要性**。被 `<strong>` 标签包裹的文本通常会以**粗体**显示，以示其重要性。

与 `<em>` 标签类似，`<strong>` 标签也具有语义意义，屏幕阅读器会以更强的语气朗读被包裹的文本。故仅需要粗体效果时，可以使用 `<b>` 标签，或者通过 CSS 来实现。

```html
<p>这是一个 <strong>着重强调</strong> 的例子。</p>
```

## 粗体，斜体，下划线

像 `<b>` 和 `<i>` 这样的标签仅用于视觉效果，而不携带任何语义信息，被称为**表现元素**。
**但值得注意的是**：最好避免使用这些标签来表示强调或重要性，因为它们不会被屏幕阅读器识别为强调或重要性（即要保持无障碍性）。

- `<b>`：表示粗体文本，但不表示重要性。
- `<i>`：表示斜体文本，但不表示强调。
- `<u>`：表示下划线文本，但不表示任何语义意义。
