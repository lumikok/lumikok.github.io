# 列表

## 无序列表

无序列表用`<ul>`标签表示，列表项用`<li>`标签表示。例如：

```html
<ul>
  <li>苹果</li>
  <li>香蕉</li>
  <li>橘子</li>
</ul>
```

## 有序列表

有序列表用`<ol>`标签表示，列表项同样用`<li>`标签表示。例如：

```html
<ol>
  <li>第一项</li>
  <li>第二项</li>
  <li>第三项</li>
</ol>
```

## 列表嵌套

列表可以嵌套使用，即在一个列表项中包含另一个列表。例如：

```html
<ul>
  <li>
    水果
    <ul>
      <li>苹果</li>
      <li>香蕉</li>
    </ul>
  </li>
  <li>
    蔬菜
    <ul>
      <li>胡萝卜</li>
      <li>菠菜</li>
    </ul>
  </li>
</ul>
```

## 描述列表

描述列表是用来表示术语及其描述的列表，使用`<dl>`标签表示，术语用`<dt>`标签表示，描述用`<dd>`标签表示。例如：

```html
<dl>
  <dt>HTML</dt>
  <dd>超文本标记语言</dd>
  <dt>CSS</dt>
  <dd>层叠样式表</dd>
  <dt>JavaScript</dt>
  <dd>编程语言</dd>
</dl>
```

**同时**,单个术语也可以有多个描述：

```html
<dl>
  <dt>HTML</dt>
  <dd>超文本标记语言</dd>
  <dd>用于创建网页的标准标记语言</dd>
</dl>
```
