# 1.HTML 基础


## 什么是 HTML


超文本标记语言（HyperText Markup Language，HTML）是一种用来创建 Web 网页的标准标记语言，用于书写网页的内容和结构。

它由一系列**元素**组成，用于定义网页中的不同部分，如标题、段落、链接、图像等，并可以对其进行格式化和布局。

### 示例



**以下示例使用的语法有：**

`<p>`（段落标签）、`<li>`（列表项标签）、`<ul>`（无序列表标签）。

```html
<p>这是一个段落。</p>
<ul>
  <li>列表项1</li>
  <li>列表项2</li>
  <li>列表项3</li>
</ul>
```

## HTML 的基本结构



一个标准的 HTML 文档通常包含以下基本结构：

```html
<!DOCTYPE html>
<html lang="en-US">
  <head>
    <meta charset="utf-8" />
    <meta name="viewport" content="width=device-width" />
    <title>My test page</title>
  </head>
  <body>
    <img src="images/firefox-icon.png" alt="My test image" />
  </body>
</html>
```

### 结构说明


- `<!doctype html>`——文档类型。**必不可少**，仅用于保证文档正常读取。
- `<html></html>`——HTML 根元素，包含整个 HTML 文档的内容。它还包含`lang`属性，用于指定文档的主要语言。
- `<head></head>`——头部元素，该元素作为想在 HTML 页面中包含但不想向用户显示的内容的容器。包括页面描述、关键字、作者信息、CSS 文件、JS 文件等。
- `<meta charset="utf-8" />`——指定文档的字符编码为 UTF-8，支持多种语言字符。
- `<meta name="viewport" content="width=device-width" />`——视口元素，确保页面在不同设备上正确显示和缩放。
- `<title></title>`——标题元素，定义网页的标题，显示在浏览器标签页上，也用于搜索引擎结果中的标题。
- `<body></body>`——主体元素，包含期望显示在网页上的所有内容，如文本、图像、链接等。

**注意**：大多数 HTML 元素都有一个**开始标签**，随后是元素内容，最后是一个**结束标签**。但是某些 HTML 元素还包含**属性**，用于提供有关元素的附加信息（例如，`<img>` 标签中的 `src` 和 `alt` 属性）。

## HTML 元素（部分）



### 嵌入图像



如基本结构示例中所示，使用 `<img>` 标签来嵌入图像：

```html
<img src="" alt="My test image" />
```

该标签有两个主要属性：

- `src`——源属性，指定图像文件的路径。
- `alt`——替代文本属性，是图像的描述内容，当图像不可见时显示该文本。

**注意**：编写的替代文本应当为读者提供足够的信息，以便他们充分理解图像所传达的内容。

**备注**：`<img>` 标签是一个自闭合标签，不需要结束标签。像这样的标签（无内容也无结束标签）被称为“**空元素**”。

### 标记文本



以下是一些常用的文本标记 HTML 元素。

#### 标题


标题元素可用于指定内容的标题或子标题。HTML 包含六个级别的标题，从 `<h1>` 到 `<h6>`。

```html
<!-- 这是一个标题示例 -->
<h1>主标题</h1>
<h2>顶层标题</h2>
<h3>子标题</h3>
<h4>次子标题</h4>
```

**备注**：上述示例中出现了**注释**，注释不会显示在网页上，但可以帮助开发者理解代码。注释的语法是 `<!-- 注释内容 -->`。

#### 段落



如上文所述，段落使用 `<p>` 标签来定义，通常用于指定常规的文本内容。

```html
<p>这是一个段落。</p>
<p>这是另一个段落。</p>
```

#### 列表



其实，web 上有大量内容都属于列表形式，比如文章目录、商品清单、导航菜单等。HTML 提供了多种列表类型。列表标记至少包含两个元素。最常见的列表元素有两种：

1. **无序列表**（Unordered List）：该列表中项目的顺序不重要，就像购物清单。用 `<ul>` 元素包装。
2. **有序列表**（Ordered List）：该列表中项目的顺序很重要，就像任务步骤。用 `<ol>` 元素包装。

每个列表项都使用 `<li>`（List Item）标签来定义。

#### 链接



链接是 HTML 中非常重要的元素，用于在网页之间导航。使用 `<a>`（Anchor）标签来创建链接。要将一些文本添加到链接中，一般需要以下几步：

1. 使用 `<a>` 标签包裹要作为链接的文本。如：

```html
<a>点击这里访问 DeepSeek</a>
```

2. 使用 `href` 属性指定链接的目标 URL。如：

```html
<a href="https://deepseek.com">点击这里访问 DeepSeek</a>
```
