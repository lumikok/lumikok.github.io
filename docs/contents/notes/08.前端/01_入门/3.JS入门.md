# JavaScript 入门

JavaScript（简称 JS）是一种为网站添加交互性的**编程语言**。交互性就是用户可以与网页进行互动，比如点击按钮、填写表单、播放视频等。以下是 JavaScript 的一些基本知识（仅简单介绍，用于快速上手）。

---

## 什么是 JavaScript

JavaScript 是一种**脚本语言**，它可以嵌入到 HTML 页面中，并由浏览器执行。与 HTML 和 CSS 不同，JavaScript 主要用于处理网页的逻辑和行为。

---

## 一个示例

下面以“Hello, World!”为例，展示如何在网页中使用 JavaScript。**注意**：一般都把 CSS 和 JavaScript 代码放在单独的文件中（可以放入特定的文件夹内），这样更方便管理和维护。

在 HTML 文件中，在结束标签 `</body>` 前添加以下代码：

```html
<script src="scripts/main.js"></script>
```

很明显，上述代码引入了一个在 `scripts` 文件夹内名为 `main.js` 的 JavaScript 文件。接下来可以在该文件内编写 JavaScript 代码：

```javascript
const myHeading = document.querySelector("h1");
myHeading.textContent = "Hello world!";
```

上述代码的作用是将网页中第一个 `<h1>` 元素的内容更改为“Hello world!”。(具体含义在后续章节中会详细介绍)

---

**注意**：为什么要把 JavaScript 代码放在 `</body>` 前？这是因为浏览器在加载网页时，会从上到下依次解析 HTML。如果 JavaScript 代码放在 `<head>` 中，可能会在 HTML 元素加载完成之前执行，导致找不到相应的元素而报错。

---

## JS 基本语法

因为 JS 是编程语言，所以它和其它编程语言是类似的（因本人已经学过了部分编程语言，故以下仅介绍 JS 的特别之处，对于基本概念不作过多介绍）。

---

### 变量

在 JS 中，一般使用 `let`来声明变量就行了（有时也会用到`const`来声明常量）。如：

```javascript
let myName = "DeepSeek";
```

此处还进行了赋值操作（不过多赘述）。

---

**注意**：JS 是弱类型语言（变量类型不需要声明，解释器会自动判断），且区分大小写。

#### 数据类型

JS 中常用的数据类型有以下几种(仅有字符串用引号括起来)：

1. 字符串（String）：表示文本数据，如 `"Hello, World!"`。
2. 数字（Number）：表示整数和浮点数，如 `42` 或 `3.14`。
3. 布尔值（Boolean）：表示真或假，只有两个值：`true` 和 `false`。
4. 数组（Array）：表示一组有序的数据集合，如 `[1, 2, 3, 4, 5]`。
5. 对象（Object）：表示键值对的集合，如 `{name: "DeepSeek", age: 25}`。

**简述对象**：JS 里的**一切都是对象**，对象能在变量中存储。（后续章节会详细介绍）

---

### 注释

和 C 和 C++一样（/_多行注释_/ 和 // 单行注释）。

### 运算符

不同于 C/C++：

1. JS 中没有`++`和`--`运算符，可以使用 `+= 1` 和 `-= 1` 来实现自增和自减操作。
2. JS 中没有`&&`和`||`运算符，可以使用 `and` 和 `or` 来实现逻辑与和逻辑或操作。
3. JS 判断相等使用 `===`，不相等使用 `!==`。

### 条件语句

与 C/C++类似，使用 `if`、`else if` 和 `else` 来实现条件判断。

### 函数

JS 中使用 `function` 关键字来定义函数。例如：

```javascript
function greet(name) {
  return "Hello, " + name + "!";
}
```

**注意**：此处`+`作字符串连接符。

### 事件

事件处理器用于响应用户的操作，如点击按钮、移动鼠标等。可以使用 `addEventListener` 方法来添加事件监听器。例如：

```javascript
document.querySelector("html").addEventListener("click", function () {
  alert("别点我！");
});
```

拆分解释：

1. `document.querySelector("html")`：将事件处理器与元素绑定，此处绑定的是整个网页。
2. `.addEventListener("click", function() {...})`：调用了 `addEventListener` 方法，传递要监听的事件类型（此处是点击事件）和事件发生时要执行的函数。
3. `alert("别点我！")`：使用 `alert` 函数弹出一个提示框，显示指定的消息。
4. `function() {...}`：定义了一个匿名函数，当点击事件发生时会执行该函数内的代码。（匿名函数即没有名字的函数，还有别的书写方式（箭头函数）： `() => {...}`）。

### 完善你的示例网站

在之前的 HTML 和 CSS 的学习过程中，我们已经创建了一个简单的示例网站。现在，我们可以通过添加一些 JavaScript 代码来增强网站的交互性。

#### 图像切换器

这里还运用了 DOM API（文档对象模型应用程序接口-后续章节会详细介绍），实现点击图像切换功能。

在 `scripts/main.js` 文件中，添加以下代码：

```javascript
const myImage = document.querySelector("img");

myImage.onclick = function () {
  const mySrc = myImage.getAttribute("src");
  if (mySrc === "images/1.png") {
    myImage.setAttribute("src", "images/2.png");
  } else {
    myImage.setAttribute("src", "images/1.png");
  }
};
```

**详解**：

1. `const myImage = document.querySelector("img");`：选择网页中的第一个 `<img>` 元素，并将其存储在变量 `myImage` 中。
2. `myImage.onclick = function() {...}`：为 `myImage` 元素添加一个点击事件处理器，当用户点击图像时会执行该函数内的代码。
3. `const mySrc = myImage.getAttribute("src");`：获取当前图像的 `src` 属性值，并将其存储在变量 `mySrc` 中。
4. `if(mySrc === "images/1.png") {...} else {...}`：判断当前图像的 `src` 属性值是否为 `"images/1.png"`。
5. `myImage.setAttribute("src", "images/2.png");`：如果条件为真，则将图像的 `src` 属性更改为 `"images/2.png"`，实现图像切换。
6. `myImage.setAttribute("src", "images/1.png");`：如果条件为假，则将图像的 `src` 属性更改为 `"images/1.png"`，实现图像切换。

#### 欢迎信息

在 `scripts/main.js` 文件中，添加以下代码：

```javascript
let myButton = document.querySelector("button");
let myHeading = document.querySelector("h1");
function setUserName() {
  let myName = prompt("请输入你的名字。");
  localStorage.setItem("name", myName);
  myHeading.textContent = "Mywebsite，欢迎你，" + myName;
}
if (!localStorage.getItem("name")) {
  setUserName();
} else {
  let storedName = localStorage.getItem("name");
  myHeading.textContent = "Mywebsite，欢迎你，" + storedName;
}
myButton.onclick = function () {
  setUserName();
};
```

**详解**：

1. `let myButton = document.querySelector("button");`：选择网页中的第一个 `<button>` 元素，并将其存储在变量 `myButton` 中。
2. `let myHeading = document.querySelector("h1");`：选择网页中的第一个 `<h1>` 元素，并将其存储在变量 `myHeading` 中。
3. `function setUserName() {...}`：定义一个名为 `setUserName` 的函数，用于设置用户的名字。
4. `let myName = prompt("请输入你的名字。");`：使用 `prompt` 函数弹出一个对话框，提示用户输入他们的名字，并将输入的值存储在变量 `myName` 中。
5. `localStorage.setItem("name", myName);`：使用 `localStorage` 对象将用户的名字存储在浏览器的本地存储中，键名为 `"name"`。
6. `myHeading.textContent = "Mywebsite，欢迎你，" + myName;`：将 `<h1>` 元素的内容更改为包含用户名字的欢迎信息。
7. `if(!localStorage.getItem("name")) {...} else {...}`：检查本地存储中是否存在键名为 `"name"` 的值。
8. `setUserName();`：如果本地存储中没有用户名字，则调用 `setUserName` 函数，提示用户输入名字。
9. `let storedName = localStorage.getItem("name");`：如果本地存储中存在用户名字，则将其取出并存储在变量 `storedName` 中。
10. `myHeading.textContent = "Mywebsite，欢迎你，" + storedName;`：将 `<h1>` 元素的内容更改为包含存储的用户名字的欢迎信息。
11. `myButton.onclick = function() {...}`：为 `myButton` 元素添加一个点击事件处理器，当用户点击按钮时会执行该函数内的代码。
12. `setUserName();`：当按钮被点击时，调用 `setUserName` 函数，允许用户重新输入他们的名字。
13. `localStorage`：Web 存储 API 的一部分，允许在用户的浏览器中存储数据，即使关闭浏览器后数据仍然存在。与 `sessionStorage` 不同，后者在浏览器会话结束时会被清除。
14. `prompt`：用于显示一个对话框，提示用户输入信息，并返回用户输入的字符串。如果用户点击“取消”，则返回 `null`。
