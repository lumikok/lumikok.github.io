# 关系选择器

## 后代选择器

**后代选择器**通常用单个空格来组合两个选择器，表示选择某个元素的所有后代元素。

```css
.box p {
  color: blue;
}
```

上面的代码表示选择所有类名为`box`的元素中的所有`<p>`元素，并将其文本颜色设置为蓝色。

## 子元素选择器

**子元素选择器**使用大于号（`>`）来组合两个选择器，表示选择某个元素的直接子元素。

```css
ul > li {
  border-top: 1px solid #ccc;
}
```

上面的代码表示选择所有`<ul>`元素的直接子元素`<li>`，并为它们添加一个顶部边框。但是**注意**：如果是附属于`<li>`的子元素（例如`<span>`），则不会被选中。

## 相邻兄弟选择器

**相邻兄弟选择器**使用加号（`+`）来组合两个选择器，表示选择紧接在某个元素之后的第一个兄弟元素。

```css
h2 + p {
  font-size: 18px;
}
```

上面的代码表示选择所有紧接在`<h2>`元素之后的第一个`<p>`元素，并将其字体大小设置为 18 像素。

## 通用兄弟选择器

**通用兄弟选择器**使用波浪号（`~`）来组合两个选择器，表示选择某个元素之后的所有兄弟元素。

```css
h2 ~ p {
  color: green;
}
```

上面的代码表示选择所有位于`<h2>`元素之后的所有`<p>`元素，并将其文本颜色设置为绿色。
