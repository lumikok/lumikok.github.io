# 属性选择器

## 存否和值选择器

有些选择器可以根据元素的属性及其值来选择元素。

| 选择器           | 示例                    | 说明                                                                            |
| ---------------- | ----------------------- | ------------------------------------------------------------------------------- |
| `[attr]`         | `a[title]`              | 选择所有具有 `title` 属性的 `<a>` 元素                                          |
| `[attr=value]`   | `input[type=text]`      | 选择所有 `type` 属性值**为** `text` 的 `<input>` 元素                           |
| `[attr~=value]`  | `div[class~=highlight]` | 选择所有 `class` 属性**包含**单词 `highlight` 的 `<div>` 元素（单词由空格分隔） |
| `[attr\|=value]` | `p[lang\|=en]`          | 选择所有 `lang` 属性值为 `en` 或以 `en-` 开头的 `<p>` 元素                      |

下面是一些示例：

```html
<h1>属性选择器示例</h1>
<ul>
    <li>Item 1</li>
    <li class="a">Item 2</
    <li class="b">Item 3</li>
    <li class="a b">Item 4</li>
```

```css
/* 选择所有具有 class 属性的 li 元素 */
li[class] {
  color: blue;
}
/* 选择 class 属性值为 "a" 的 li 元素 */
li[class="a"] {
  font-weight: bold;
}
/* 选择 class 属性包含单词 "b" 的 li 元素 */
li[class~="b"] {
  text-decoration: underline;
}
/* 选择 class 属性值以 "a" 开头的 li 元素 */
li[class|="a"] {
  background-color: yellow;
}
```

## 子字符串匹配选择器

除了上述选择器外，还有一些选择器可以根据属性值的子字符串进行匹配：

| 选择器          | 示例                     | 说明                                                               |
| --------------- | ------------------------ | ------------------------------------------------------------------ |
| `[attr^=value]` | `a[href^="https"]`       | 选择所有 `href` 属性值**以** `https` 开头的 `<a>` 元素             |
| `[attr$=value]` | `img[src$=".png"]`       | 选择所有 `src` 属性值**以** `.png` 结尾的 `<img>` 元素             |
| `[attr*=value]` | `div[data-info*="user"]` | 选择所有 `data-info` 属性值**包含**子字符串 `user` 的 `<div>` 元素 |

总结来说呢：`^`是开头，`$`是结尾，`*`是包含。

## 大小写敏感

默认情况下，属性选择器对属性值的匹配是**区分大小写**的。不过，可以通过添加修饰符`i`（不区分大小写）或`s`（区分大小写）来改变这一行为。例如：

```css
/* 不区分大小写 */
a[href="example.com" i] {
  color: red;
}
/* 区分大小写 */
a[href="Example.com" s] {
  color: blue;
}
```
