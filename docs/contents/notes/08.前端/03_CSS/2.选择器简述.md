# 选择器简述

选择器是 CSS 中用于选取 HTML 元素以应用样式的模式。通过选择器，开发者可以指定哪些元素应该受到特定样式规则的影响。选择器有多种类型，每种类型都有其独特的语法和用途。

## 选择器列表

当有多个使用相同样式的 CSS 选择器时，可以将它们组合在一起，以逗号分隔。例如：

```css
h1,
h2,
h3 {
  font-family: Arial, sans-serif;
  color: blue;
}
```

## 选择器的种类

### 基础选择器

- **元素选择器**：选择特定类型的 HTML 元素。例如，`p`选择所有段落元素。
- **类选择器**：选择具有特定类属性的元素。类选择器以点号（`.`）开头。例如，`.highlight`选择所有具有`highlight`类的元素。
- **ID 选择器**：选择具有特定 ID 属性的元素。ID 选择器以井号（`#`）开头。例如，`#header`选择具有`header` ID 的元素。
- **通用选择器**：选择所有元素。通用选择器使用星号（`*`）。例如，`*`选择所有 HTML 元素。
- **属性选择器**：选择具有特定属性的元素。例如，`[type="text"]`选择所有类型为文本的输入元素，`a[title]`选择所有具有`title`属性的链接元素。
- **伪类选择器**：选择处于特定状态的元素。例如，`:hover`选择鼠标悬停时的元素，`:first-child`选择作为其父元素第一个子元素的元素。

### 运算符

- **后代选择器**（空格）：选择某元素内部的所有指定子元素。例如，`div p`选择所有位于`div`元素内的`p`元素。
- **子选择器**（`>`）：选择某元素的直接子元素。例如，`ul > li`选择所有作为`ul`直接子元素的`li`元素。
- **相邻兄弟选择器**（`+`）：选择紧接在某元素之后的兄弟元素。例如，`h1 + p`选择紧跟在`h1`之后的第一个`p`元素。
- **通用兄弟选择器**（`~`）：选择某元素之后的所有兄弟元素。例如，`h1 ~ p`选择所有在`h1`之后的`p`元素。
- **伪元素选择器**（`::`）：选择元素的特定部分。例如，`p::first-letter`选择段落的第一个字母，`p::before`在段落内容前插入内容。
