# 伪类和伪元素

## 何为伪类

伪类是选择器的一种，它用于选择处于特定状态的元素。它们的表现就类似于类选择器，但伪类并不是在 HTML 中定义的类，而是由 CSS 定义的。 伪类通常以冒号（:）开头，比如`:hover`。

### 简单伪类

- `:nth-child(n)`：选择父元素的第 n 个子元素。
- `:first-child`：选择父元素的第一个子元素。
- `:last-child`：选择父元素的最后一个子元素。
- `:only-child`：选择唯一的子元素。
- `:invalid`：选择无效的表单元素。
- `:not(selector)`：选择不匹配指定选择器的元素。
- `:disabled`：选择被禁用的表单元素。
- `:enabled`：选择可用的表单元素。
- `:empty`：选择没有子元素的元素。
- `:root`：选择文档的根元素，通常是`<html>`元素。
- `:lang(language)`：选择具有特定语言属性的元素。

比如：

```html
<article>
  <p>这是第一段。</p>
  <p>这是第二段。</p>
  <p>这是第三段。</p>
</article>
```

```css
article p:first-child {
  color: red;
}
article p:nth-child(2) {
  color: blue;
}
```

上面的代码会将第一段文字变为红色，第二段文字变为蓝色。

### 动态伪类

以下的伪类主要用于处理用户与网页元素的交互：

- `:hover`：当用户将鼠标悬停在元素上时应用样式，一般用于链接和按钮。
- `:active`：当用户点击并按住元素时应用样式。
- `:focus`：当元素获得焦点时应用样式，通常用于表单输入字段。
- `:visited`：选择用户已经访问过的链接。
- `:link`：选择用户未访问过的链接。
- `:checked`：选择被选中的复选框或单选按钮。

例如：

```html
<a href="#">这是一个链接</a>
```

```css
a:link,
a:visited {
  color: rebeccaPurple;
  font-weight: bold;
}

a:hover {
  color: hotpink;
}
```

上面的代码会将链接设置为紫色和加粗，当鼠标悬停时变为粉色。

## 何为伪元素

伪元素以类似伪类的方式工作，但它们用于创建和样式化不存在于文档树中的虚拟元素。伪元素通常以双冒号（::）开头，以区分伪类。

### 常用伪元素

- `::before`：在元素内容之前插入的内容。(注意：需要设置`content`属性才能显示)
- `::after`：在元素内容之后插入的内容。(同上)
- `::first-letter`：选择元素的第一个字母。
- `::first-line`：选择元素的第一行文本。
- `::placeholder`：选择输入字段的占位符文本。
- `::selection`：选择用户选中的文本。
- `::marker`：选择列表项的标记符号（如项目符号或数字）。
- `::backdrop`：选择全屏模式下的背景。
- `::cue`：选择视频字幕中的文本。

例如：

```html
<p class="example">这是一个示例段落。</p>
```

```css
.example::before {
  content: "注意：";
  color: red;
  font-weight: bold;
}
.example::after {
  content: "（结束）";
  color: blue;
  font-style: italic;
}
```

上面的代码会在段落前添加“注意：”并将其设为红色加粗，在段落后添加“（结束）”并将其设为蓝色斜体。

## 伪类和伪元素组合使用

有时，我们可以将伪类和伪元素结合使用，以实现更复杂的样式效果。例如：

```css
article p:first-child::first-line {
  font-size: 120%;
  font-weight: bold;
}
```

上面的代码会将文章中第一个段落的第一行文字设置为 120%的字体大小和加粗效果。

### 特殊的伪元素

在上述中其实已经提到了一组比较特殊的伪元素，它们分别是`::before`和`::after`。它们与`content`属性结合使用，可以在元素的内容前后插入额外的内容。这在创建**装饰性效果**或**添加图标**时非常有用。
