# 位运算

以下为C语言位运算部分内容。

## 六大位运算符

### 按位与 &（掩码）

按位与运算符 & 会对两个整数的每一位进行比较，只有当两个对应的二进制位都为1时，结果位才为1，否则为0。

```c
//  === 四大核心应用 ===

// 1.检查特定位是否为1（掩码测试）
#define CHECK_BIT_3(num) ((num) & 0x08) // 检查第3位是否为1

// 2.清除特定位（掩码清零）
int clear_bit_2(int num) {
    return num & ~0x04; // 清除第2位
}

// 3.提取部分位（位字段提取）
int extract_bits_1_to_3(int num) {
    return (num >> 1) & 0x07; // 提取第1到第3位
}
